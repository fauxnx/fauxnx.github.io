Project Details

Titan Engine

A graphics engine I made along with my friend Gonzalo Veiras. It's built using modern C++ and OpenGL.





Features

Deferred Rendering

Titan Engine uses deferred rendering to allow the use of multiple postprocesses, as well as track the state of the pipeline at any stage.


PBR

Titan Engine uses a PBR model based on the Filament Engine. Materials are lit using physically-correct equations, ensuring conservations of energy and offering realistic results.    Hard surface materials, such as metals and plastics, are lit using a combination of the Cook-Torrance BRDF for the specular lobe, and a Burley BRDF for the diffuse.
